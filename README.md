## Cloudflare Pages

- Build: `npm run build`
- Output dir: `out`
- Env:

```
NEXT_PUBLIC_SITE_URL=https://<your>.pages.dev
NEXT_PUBLIC_WHATSAPP_NUMBER=27000000000
FORMSPREE_ENDPOINT=
```
